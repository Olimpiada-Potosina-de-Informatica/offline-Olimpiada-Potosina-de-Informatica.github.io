Operación Matemática
Una operación es la aplicación de un operador sobre un elemento o elementos de un conjunto dado. Aplicando ciertas reglas.

Operador Matemático
Un operador es un símbolo matemático que indica que operación se realizara sobre los operandos (elementos a aplicar).

Una vez realizadas las definiciones, vamos a delimitar este tema, para empezar vamos a hablar acerca de las operaciones aritméticas, hay varios tipos de operaciones pero nos vamos a centrar en las aritméticas por ahora. Estas son las primeras que vemos en nuestra educación, de seguro ya te imaginaras sobre que vamos a hablar.

¡Empecemos! Veamos las operaciones más básicas, la Suma y Resta, después la Multiplicación y División, posteriormente las más complejas como la Potenciación, Radicación, Módulo y la Logaritmación.

Suma y Resta

La operación de la suma consiste en combinar los valores de dos o más elementos en uno. Las variables A y B son sumados y la variable C suma.

A + B = C

Además, posee algunas propiedades como: Asociativa, Conmutativa, Elemento Neutro y Elemento Opuesto.

La operación de la Resta podemos verla como la operación inversa de la suma, donde el elemento A es minuendo y el B sustraendo y al C diferencia.

A - B = C

Podemos ver que la resta no es conmutativa.

Multiplicación y División

La operación de la Multiplicación consiste en realizar repetidas veces la suma de un factor tantas como el otro factor la indique, en otras palabras se sumara el factor A, tantas como lo indica el factor B y el resultado de esto, C le llamaremos producto.

A * B = C

Además, posee algunas propiedades al igual que las operaciones anteriores, como: Asociativa, Conmutativa, Distributiva, Elemento Neutro y Elemento Unitario o tambien llamado Elemento Inverso, ¡Eso te toca a ti!

La operación de la División determina cuantas veces cabe un elemento en otro; el elemento A (Dividendo) es el que es partido en B partes (Divisor), y el elemento C (Cociente) menciona cuantas veces cabe B en A

A / B = C
