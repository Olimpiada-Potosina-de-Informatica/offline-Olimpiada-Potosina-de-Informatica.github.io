Bloque I - Introducción Matemática

Potenciación, Radicación, Módulo, Logaritmación y sus propiedades

Operación Matemática
Una operación es la aplicación de un operador sobre un elemento o elementos de un conjunto dado. Aplicando ciertas reglas.

Operador Matemático
Un operador es un símbolo matemático que indica que operación se realizara sobre los operandos (elementos a aplicar).

Esta es una extensión del tema anterior, donde se verán las operaciones aritméticas que faltaron ver en el tema anterior, se verá la Potenciación, Radicación, Módulo y la Logaritmación.

Potenciación

Es la operación en la cual la cantidad llamada base (A) se debe multiplicar por ella misma las veces que lo indique el exponente (B). De lo anterior se puede ver como una multiplicación con factores idénticos, donde el número de factores es la potencia que se tiene.

Radicación

Es la operación que permite hallar un valor que multiplicado tantas veces como lo indica el índice, dé el valor que se encuentra dentro del radical, el cual recibe el nombre de radicando.Donde A es la base, B el exponente y C el índice. Logaritmación

El logaritmo con base b de un número A, es el exponente a, al cual se eleva la base b para obtener el resultado o argumento A dando como resultado C, además, este se puede representar como en la siguiente ecuación.

Los logaritmos de Briggs son un logaritmo con el que se trabaja con regularidad usando la base 10, por lo que regularmente no se escribe la base; cabe mencionar que se le llama logaritmo natural al logaritmo que utiliza la base e = 2.718281.

El logaritmo de cualquier número está formado por una parte que corresponde a un número entero llamado característica y otro decimal que recibe el nombre de mantisa.
